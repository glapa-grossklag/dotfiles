# dotfiles

A collection of my dotfiles for:

* Zsh
* Vim & NeoVim
* i3
* GNU Make
* Zathura

## Installation

```sh
git clone git@github.com:glapa-grossklag/dotfiles.git
bash ./install.sh
```

Note: `install.sh` does not install any of the programs, please use your
package manager of choice to do so.
